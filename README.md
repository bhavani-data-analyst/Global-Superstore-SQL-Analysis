# Global Superstore SQL Analysis

## Project Overview

This project demonstrates SQL-based data analysis using the Global Superstore dataset. The analysis was performed in SQL Server Management Studio (SSMS) to generate business insights from sales, profit, customers, categories, and regions.

## Tools Used

- SQL Server
- SQL Server Management Studio (SSMS)
- Global Superstore Dataset

## SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- TOP
- DISTINCT
- COUNT
- SUM
- AVG
- MAX
- MIN
- GROUP BY
- HAVING
- CASE WHEN
- Date Functions
- Views

## Business Questions Solved

### Customer Analysis
- Top customers by sales
- Top customers by profit

### Region Analysis
- Region-wise sales analysis
- Region-wise profit analysis

### Category Analysis
- Sales by category
- Profit by category

### Sales & Profit Analysis
- Highest sales value
- Lowest profit value
- Average profit analysis

## Project Outcome

This project helped develop practical SQL skills for data cleaning, data exploration, aggregation, filtering, and business reporting.
